# Пример: prisms-library.md

> **Что это:** Файл для переноса призм на следующий товар.  
> Создаётся автоматически Skill 2 в конце каждого прогона (раздел «Файл призм для переноса»).  
> Передай этот файл в следующий прогон Skill 2 для режима «с багажом».

---

## Шаблон

Скопируй блок ниже — он будет заполнен Skill 2 автоматически.

```
product_key: [ключ предыдущего товара]
category: [категория]
date: [дата прогона]

ПРИЗМЫ:

### [Название призмы 1]
уверенность: ●/◐/○
репрез.: ~N%
мотивация: [1–2 предложения]
критичные сигналы: [список]
опасные сигналы: [список]
предубеждения: [список]
оффер: [1 предложение]
запреты: [список]

...

ЭВРИСТИКИ КАТЕГОРИИ:
- [стереотип 1]
- [стереотип 2]

АНТИКАТЕГОРИЯ:
[описание]

НЕЛЬЗЯ ОБЕЩАТЬ:
- [запрет 1]
- [запрет 2]
```

## Пример заполненного файла

*(На основе парфюмированного крема для рук — категория «уход за руками»)*

```
product_key: repose-flower-hand-cream-v1
category: уход за руками (парфюмированные кремы)
date: 2025-04-01

ПРИЗМЫ:

### Призма: Дарящий
уверенность: ●
репрез.: ~22%
мотивация: Покупает для другого человека. Главный критерий — «будет ли приятно получить в подарок». Оценивает: выглядит ли дорого, понятно ли что внутри, не стыдно ли вручить.
критичные сигналы: 
  - упаковка (внешний вид, коробка, тюбик)
  - аромат (первое впечатление при открытии)
  - ощущение «премиальности»
опасные сигналы:
  - смятая/повреждённая упаковка при доставке
  - слишком резкий или непривычный аромат
  - ощущение «дешёвого продукта за дорогую цену»
предубеждения:
  - красивая упаковка = хорошее содержимое
  - крем для рук = безопасный подарок (не слишком интимный, не слишком банальный)
  - дорогой = оригинальный
оффер: Крем, который выглядит как подарок из бутика — и работает так же хорошо.
запреты:
  - не использовать слово «бюджетный» или намёки на экономию
  - не показывать продукт без упаковки
  - не упоминать, что упаковка может деформироваться при доставке

### Призма: Уходовый рационалист
уверенность: ●
репрез.: ~31%
мотивация: Покупает для себя, ищет реальный результат для кожи. Скептичен к маркетингу, 
  читает отзывы на предмет «а работает ли это на самом деле». Интересует состав, 
  стойкость эффекта, соотношение цена/объём.
критичные сигналы:
  - увлажнение (насколько, как долго)
  - текстура (впитывается / не липнет)
  - состав (натуральные компоненты, отсутствие парабенов)
опасные сигналы:
  - «маркетинговый» аромат без реальной пользы
  - «не работает через неделю»
  - слишком дорого за маленький объём
предубеждения:
  - цветочные кремы = только запах, без реального ухода
  - «парфюмированный» = агрессивная химия
  - дорогое ≠ лучше (надо проверять)
оффер: Ухаживает серьёзно — увлажняет 8–12 часов, впитывается за 2 минуты, без остатка.
запреты:
  - не обещать «24 часа» без доказательств
  - не делать акцент на аромате как главном свойстве
  - не использовать слова «магия», «волшебный», «невероятный»

### Призма: Аромат как опыт
уверенность: ◐
репрез.: ~18%
мотивация: Для этого человека запах — главная ценность. Использует крем как часть 
  ароматного ритуала. Сравнивает с духами. Ценит шлейф и стойкость.
критичные сигналы:
  - стойкость аромата (после нанесения, после мытья рук)
  - характер запаха (цветочный, пудровый, свежий)
  - аромат не меняется при смешении с запахом рук
опасные сигналы:
  - «запах исчезает через час»
  - «химический» или «мыльный» аромат
  - аромат слишком резкий / «на любителя»
предубеждения:
  - хороший аромат = дорогой аромат
  - крем для рук = «недопарфюм» (можно только если аромат правда хорош)
  - стойкость = качество продукта
оффер: Парфюмированный ритуал для рук: цветочный шлейф держится 4–6 часов даже 
  после мытья.
запреты:
  - не использовать расплывчатое «нежный аромат»
  - не игнорировать тему стойкости в описании
  - не смешивать аромат с «лечебными» ноутами

### Призма: Экономный практик
уверенность: ◐
репрез.: ~15%
мотивация: Смотрит на объём, расход, цену за мл. Подозревает переплату за бренд. 
  Ищет доказательство, что продукт «стоит своих денег».
критичные сигналы:
  - объём (мл)
  - расход (хватает ли надолго)
  - реальный уходовый эффект (не исчезает через день)
опасные сигналы:
  - «маленький объём за большие деньги»
  - «закончился слишком быстро»
  - ощущение, что платишь за упаковку
предубеждения:
  - красивая упаковка = переплата за маркетинг
  - парфюмированный = дороже, не лучше
  - известный бренд = завышенная цена
оффер: 75 мл хватает на 3 месяца ежедневного использования — считай сам.
запреты:
  - не прятать объём в характеристики
  - не делать акцент на упаковке (воспринимается как «переплата»)
  - не сравнивать с дорогими брендами без обоснования

### Призма: Молчаливое большинство
уверенность: ◐ (структурная)
репрез.: ~80–90% покупателей
мотивация: Купили, использовали, не написали ничего. Не разочарованы и не восхищены. 
  Продукт «работает» — и этого достаточно.
критичные сигналы: нет явных (именно поэтому молчат)
опасные сигналы:
  - что-то неожиданно плохое (не то, что ожидали — заставит написать отзыв)
  - обман: обещали X, получили не-X
предубеждения: стандартные для категории
оффер: нет (молчат — значит, всё нормально)
запреты:
  - не создавать завышенных ожиданий, которые нельзя выполнить

ЭВРИСТИКИ КАТЕГОРИИ:
- Мятая упаковка → подозрение в подделке (упаковка = сигнал оригинальности)
- Резкий аромат при открытии → «слишком химический», даже если состав натуральный
- Дорогая цена + красивая упаковка → «переплачиваю за бренд»
- «Натуральный» в названии → безопаснее, нежнее, для чувствительной кожи
- Маленький объём → «невыгодно», даже если концентрация высокая
- Белый / пастельный цвет упаковки → «аптечный», «клинически чистый»
- Французские слова в названии → «профессиональная косметика»

АНТИКАТЕГОРИЯ:
Обычный аптечный крем (Neutrogena, Атодерм) — функциональный, без эмоций, 
«лечит а не балует». Наш продукт — противоположность: не лечение, а ритуал. 
Использовать в карточке: «не просто крем — [образ]».

НЕЛЬЗЯ ОБЕЩАТЬ:
- «24 часа увлажнения» без условий — не подтверждается отзывами
- «Полностью натуральный состав» — в составе есть синтетические ароматизаторы
- «Аромат держится весь день» — стойкость сильно варьируется между покупателями
- «Подходит для чувствительной кожи» — есть отзывы о реакции у чувствительных
```
