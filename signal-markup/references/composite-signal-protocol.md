# Составные сигналы

> Все правила и примеры включены в основной промпт: [../prompts/extract-signals.md](../prompts/extract-signals.md)
>
> Этот файл — краткая справка. Не нужно читать его отдельно.

## Два правила

1. **Можно разрезать** (каждая часть имеет свой смысл) → два отдельных сигнала
2. **Нельзя разрезать** (смысл зависит от обеих частей) → один сигнал, категория по факту (результат > ожидание)

## Примеры

- «Хотела лёгкую текстуру, но крем жирный» → ⚪ + 🔴 (разрезаем)
- «Беру дочке, она в восторге» → 🟠 + 🟢 (разрезаем)
- «Думала запах мягче, а он резкий» → 🔴 целиком (не разрезаем)
