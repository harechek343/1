# Формат входных данных

> Все правила и примеры включены в основной промпт: [../prompts/extract-signals.md](../prompts/extract-signals.md)
>
> Этот файл — краткая справка. Не нужно читать его отдельно.

## Основной формат: Markdown-таблица

| Поле | Обязательно | Описание |
|------|-------------|---------|
| `#` | да | unit_index — уникальный номер |
| `Имя` | да | Имя покупателя |
| `Текст` | да | Текст отзыва |
| `Дата` | нет | Дата отзыва |
| `Звёзды` | нет | Оценка 1–5 |

## Если вход — не таблица

Сырые дампы с Wildberries, Ozon, HTML — модель сама нумерует отзывы (#1, #2, #3…), извлекает имена и текст, исключает ответы продавца.

**НЕ писать скрипт. Читать и нумеровать вручную.**
