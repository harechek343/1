## Быстрые правила

| Случай | Решение |
|--------|---------|
| Несколько «Покупателей» | Различать по #номеру, не по имени |
| «Супер!» без конкретики | Мусор (Блок D) |
| Подходит в 2 категории | Попробуй разрезать → не получилось → ручной разбор |
| 1 звезда + позитив в тексте | Оценивать текст, не звёзды |
| «Не то» без X | Негатив или ручной разбор |
| «Как X» с конкретным X | Сравнение |
| «Как X» без конкретного X | Не сравнение |
| Один сигнал повторяется 3 раза | Записать ОДИН раз |
| Вопрос об использовании | Сценарий |
| Вопрос «подойдёт ли?» | Сомнение |
